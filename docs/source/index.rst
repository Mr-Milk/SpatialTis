Welcome to spatialTis's documentation!
==========================================

spatialtis is a data analysis toolkit for multiplexed tissue data.
Applied to data generated by following technology:

- Imaging Mass Cytometry
- Multiplex Ion Beam Imaging
- etc.

-------------

.. Note:
